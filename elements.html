KeiCMS HTML Theme

Responsive Navigation

  • Navigation is hidden on < 768px with toggleable #mobilenav-icon.
  • Sticky-to-top positioning.
  • Dropdown links created in an Ordered List inside a .dropdown, <nav> List Item.

Responsive Sidebar

  • <aside> with Unordered List for links.
  • For tabbed content links: javascript:toggleTab(this, 'contentElementID'); <section id="contentElementID" class="tabbedContent">
  • Links are displayed as full-width when width is < 768px.
  • <main> width is overwritten with --aside-width.

Global Layout

Header

    <header>

  • Defines a header for a document or section.

      • <nav>

    • Defines navigation links.
    • Mobile responsiveness w/ Icon toggle.
    • Stick-to-top positioning.
    • Hoverable dropdown links.

      <div class="banner">

    • Styling between header and content.

Footer

    <footer>

  • Defines a footer for a document or section.
  • Optional unordered list for left-aligning links.

Content

    Optional1: <aside>

  • Defines content aside from the page content.
  • Place before <main> to work.
  • Variable width using --aside-width (changes <main> margin)
  • Links sent via JavaScript to toggle .tabbedContent: javascript:toggleTab(event, 'tabElementID');

    <main>

  • Specifies the main content of a document.
  • Variable width set using --main-width (unless optional1 present)
  • Has a .padded class for styling.

      • <section>

    • Defines a section in a document.

      <article>

    • Defines an article.
